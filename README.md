# Student Performance Prediction System

## Overview
An AI-powered system using machine learning to predict student performance and identify at-risk students for early intervention.

## Features
- ✅ 93.1% prediction accuracy (XGBoost)
- ✅ 94% recall rate
- ✅ Real-time predictions via web interface
- ✅ Personalized recommendations
- ✅ Multiple algorithms (Random Forest, XGBoost, ANN)

## Quick Start

### Prerequisites
- Python 3.8+
- pip
- Virtual environment (recommended)
